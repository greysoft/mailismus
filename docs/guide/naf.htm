3 - NAF

NAF (Network Application Framework) is the foundation layer upon which Mailismus is built.
It is bundled with Mailismus under the same GNU Affero licence, but is also available separately as an open source package.

You don't need to know much about NAF in order to use Mailismus, but it's Guide is included here if you wish to refer to it.

From the point of view of a Mailismus user, the naf.xml config file will mainly be used to enable or disable Mailismus components such as IMAP, POP3 or SMTP, according to your requirements.
