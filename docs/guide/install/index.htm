2 - Installation

After downloading the install package, follow the Unix or Windows instructions (see menu on left) as appropriate, to install it on your system.

Having gotten the basic Mailismus setup working, you may want to enable its database functionality.
This is an optional feature which is introduced in §4.11, so follow the SQL link on the left if you wish to proceed.
